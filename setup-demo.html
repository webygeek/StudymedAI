Setup Demo Accounts

⚠️ Important: Supabase Requires Manual Setup

Due to email confirmation requirements, you need to manually create accounts in Supabase Dashboard first.

Quick Steps:

  1. Go to Supabase Dashboard → Authentication → Settings
  2. Disable "Enable email confirmations"
  3. Go to Authentication → Users → "Add User"
  4. Create: student@demo.com / demo123456 (check "Auto Confirm")
  5. Create: admin@demo.com / admin123456 (check "Auto Confirm")
  6. Then click button below to add profiles

Or use the signup form:

After disabling email confirmation, you can create accounts directly from the login page using the "Sign Up" tab.
