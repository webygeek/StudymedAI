## Features to Test

### Student Dashboard
- View study statistics
- Weekly progress chart
- Today's schedule preview
- Quick access to all features

### Study Session
- Select subject and topic
- Start timed study session
- Take notes during session
- Sessions saved to database

### Assessment
- Take practice quizzes
- View score and results
- Review answers with explanations
- Results saved to database

### Memory Room
- View flashcards (click to flip)
- Browse mnemonics
- Create new flashcards
- Flashcards saved to library

### AI Tools
1. **Clinical Case Simulator** - Diagnose patient cases
2. **Drug Chart Creator** - Compare medications
3. **OSCE Practice Partner** - Clinical exam checklist
4. **Differential Diagnosis** - Generate DDx lists
5. **Exam Notes Condensor** - Summarize long notes
6. **Audio to Notes** - Convert recordings to notes

### Schedule
- View upcoming study sessions
- Add new schedule items
- Sessions saved to database

### My Library
- View all saved content
- Filter by type (flashcards, notes, etc.)
- Delete items
